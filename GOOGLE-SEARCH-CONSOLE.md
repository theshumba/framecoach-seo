# Google Search Console Setup — CRITICAL

This is the single most important free SEO tool. Do this FIRST.

## Step 1: Verify framecoach.io
1. Go to https://search.google.com/search-console
2. Add property → URL prefix → https://framecoach.io
3. Verify via DNS TXT record (ask co-founder) or HTML file upload

## Step 2: Verify the blog
1. Add property → URL prefix → https://theshumba.github.io/framecoach-blog
2. Verify via HTML file (already set up for GitHub Pages)

## Step 3: Submit Sitemaps
- For framecoach.io: Submit sitemap URL (ask co-founder to add one)
- For the blog: Submit https://theshumba.github.io/framecoach-blog/sitemap.xml

## Step 4: Request Indexing
- After blog goes live, use "URL Inspection" tool
- Paste each hub page URL and click "Request Indexing"
- Do this for:
  - https://theshumba.github.io/framecoach-blog/
  - https://theshumba.github.io/framecoach-blog/camera-settings-guide/
  - https://theshumba.github.io/framecoach-blog/learn-filmmaking/
  - https://theshumba.github.io/framecoach-blog/filmmaker-tools/

## Step 5: Monitor (Weekly)
- Check "Performance" tab for which queries bring impressions
- Feed those queries back into blog content
- Check "Coverage" for any indexing errors
